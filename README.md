Создана база данных PostgreSQL (через Docker) для хранения задач и категорий через SeaORM.  
Использование сущностей в функциях для создания задачи, создания категории, их удаления, просмотра и отметки выполненной. 
Управление осуществляется через консольное меню, которое было реализовано с помощью match.
Работа с асинхронностью (Tokio), использованы CRUD операции SeaORM, работа с окружением и безопасность с помощью Dotenvy и .env, использованы коллекции и функции для обработки пользовательского ввода.
